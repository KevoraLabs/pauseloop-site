PauseLoop Support

Need help with focus sessions, reminders, or break screens? Start here.

Common checks

  • Open PauseLoop settings from the menu bar icon to adjust focus, break, reminder, and system options.
  • If notifications are unavailable, open macOS System Settings and allow notifications for PauseLoop.
  • If break screens or timers feel unexpected, check your active schedule and idle detection settings.

Report an issue

For bugs, questions, or feature requests, please use the KevoraLabs project page. Include your macOS version, PauseLoop version, and what you expected to happen.
